## reversi2
* モード  
    - カスタムサイズ  
        `--size [width] [height]`  
        で盤面のサイズを変更できます。ただし2以上の値に限ります。

    - 自動対戦  
        `--auto [先手:  1(auto) or 0(manual)] [後手: 1(auto) or 0(manual)]`
        で、どちらを自動にするか決められます。自動は、ただの乱数です。

    - 盤面無しモード  
        `--hidden`  
        で、盤面の表示をなくすことが出来ます。自動対戦でとても大きな盤面で行う場合、描画時間を短縮する為です。

    - カスタム結果表示  
        `--result [milisecond]`  
        で、通常結果を表示するのにアニメーションで1枚ずつ表示されますが任意の長さに変更できます。
* 離脱  
    試合を途中で終了したい場合、(-99, -99)の位置にコマを置くよう指示することで可能です。
